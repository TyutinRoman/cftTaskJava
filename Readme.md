Readme
 
Проект InsertionSort предназначен для сортировки методом вставок двух видов входных данных: строк и целых чисел.

Для запуска программы необходим установленный JRE.
Запуск производистся из коммандной строки.

Пример комменды для запуска:
java -jar InsertionSort.jar 1.txt 2.txt -s -d (Сортировка строковых данных из файла 1.txt по убыванию, и запись в файл 2.txt)

Для выполнения программы нужно ввести 4 аргумента:
Первый аргумент:"1.txt" - файл, из которого будет производиться чтение. (!Файл должен содержать целые числа или строки, запись элементов происходит один над другим, файл содержащий целые числа не должен содержать пробелов, иначе программа прекратит работу, отразив ошибку "Тип данных элементов входного файла не соответствует заявленному".)  

Второй аргумент:"2.txt" - файл, в который будет производиться запись отсортированных данных.
(! Расширение считываемого и записываемого файлов должно быть ".txt", иначе выполнение программы не будет возможно, что будет отражено соответствующей ошибкой.)
(Названия считываемого и записываемого файлов могут быть любыми.)

Третий аргумент:"-i" - для сортировки целых чисел, либо "-s" - для сортировки строк.

Четвертый аргумент:"-a" - для сортировки по возрастанию, либо "-d" - для сортировки по убыванию.


Если командная строка открыта для папки в которой находятся программа, входной текстовый документ, то писать путь до программы и документа необязательно.
Если программа и документ находятся в других папках (второй случай), то следует записать полный путь до программы и документа. Выходной документ можно сохранить в любой папке. 
Пример: "java -jar D:\InsertionSort.jar D:\Desktop\3.txt С:\4.txt -i -d" (Программа и считываемый находятся на диске D, а запись производится на диск С)
Рекомендуется запуск коммандной строки от имени администратора.
